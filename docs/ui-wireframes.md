# UI Wireframes

This document describes the layout and design of all pages in the Digital Meeting Assistant application.

## Design Principles

- **Clean and Modern**: Minimalist design with clear visual hierarchy
- **Responsive**: Works on desktop, tablet, and mobile devices
- **Accessible**: Clear labels, good contrast, keyboard navigation
- **Consistent**: Uniform styling and component patterns throughout

**Features:**
- Filter buttons at top (All, Pending, In Progress, Completed, Overdue)
- Grid layout of task cards
- Each card shows: title, deadline, status, priority
- Mark complete button for pending/in-progress tasks
- Color-coded status badges
- Priority indicators

## Component Specifications

### Meeting Card
- Title (bold, larger font)
- Date/time with calendar icon
- Location with location icon
- Status badge (color-coded)
- Organizer information
- Action buttons (View Details, View QR Code)

### Task Card
- Title (bold)
- Description (if available)
- Deadline with calendar icon
- Status badge
- Priority badge
- Assigned user information
- Mark complete button
- Overdue indicator (red border if overdue)

### QR Display
- Large QR code image (centered)
- Token display (if needed)
- Instructions text
- Download/print options (future enhancement)

### Summary Panel
- Section headers with underlines
- Summary text (readable, well-spaced)
- Action items list (numbered)
- Full minutes (monospace font, scrollable)

## Responsive Design

### Mobile (< 768px)
- Sidebar becomes hamburger menu
- Single column layout
- Stacked statistics cards
- Full-width forms
- Touch-friendly button sizes

### Tablet (768px - 1024px)
- Sidebar can be collapsible
- Two-column layouts where appropriate
- Adjusted card sizes

### Desktop (> 1024px)
- Full sidebar always visible
- Multi-column layouts
- Optimal spacing and sizing

## Color Scheme

- **Primary**: #3498db (Blue)
- **Success**: #27ae60 (Green)
- **Warning**: #f39c12 (Orange)
- **Danger**: #e74c3c (Red)
- **Background**: #f5f5f5 (Light Gray)
- **Text**: #2c3e50 (Dark Gray)
- **Borders**: #ddd (Light Gray)

## Typography

- **Headings**: Bold, larger sizes (1.5rem - 2rem)
- **Body**: Regular, readable (1rem)
- **Labels**: Medium weight (500)
- **Code/Monospace**: Courier New for transcripts/minutes
